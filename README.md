# Rewind regex Java

This is a java version of an attempt at faster regular expressions, expanding on the knowledge of building [std.regex](https://dlang.org/phobos/std_regex.html) for the D language.
And as a companion to upcomming [rewind-regex for D](https://github.com/DmitryOlshansky/rewind-regex) some techniques are imposible in Java land but I'm optmistic that it can still beat the competition.
